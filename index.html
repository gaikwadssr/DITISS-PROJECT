Project Name

AWS Web Server Deployment with CI/CD, Prometheus Monitoring, and IPS Security using Snort

Project Description

This project focuses on automating the deployment of a web server using a CI/CD pipeline while integrating real-time monitoring and intrusion prevention for enhanced security. The infrastructure is distributed across multiple virtual machines (VMs), each serving a dedicated role to ensure modularity, efficiency, and scalability. The deployment process is automated using Jenkins, with Git for version control and SonarQube for code quality analysis. Docker is utilized for containerized application deployment. Prometheus and Grafana are integrated for real-time monitoring and visualization, ensuring system health and performance tracking. For network security, Snort is deployed as an Intrusion Prevention System (IPS) to detect and actively block potential threats, ensuring a secure infrastructure. This project establishes a streamlined, secure, and continuously monitored web application deployment, optimizing both performance and security within a cloud-based environment.
